# Cursor Landing Page Clone

This project is a recreation of the [Cursor.com](https://cursor.com) landing page, built to closely match the original design and responsiveness.

## Recreated Sections

The following sections have been implemented with high attention to detail:

- **Navigation Header**: Responsive navigation with hover effects.
- **Hero Section**: Main value proposition and primary call-to-action.
- **Social Proof**: "Trusted by" section with company logos.
- **Features**: Detailed feature blocks for Agent, Autocomplete, and Ecosystem.
- **Testimonials**: Grid of testimonials from industry leaders.
- **Use Cases & Stories**: specific use cases and integration stories.
- **Changelog**: A timeline view of recent updates.
- **Team**: Applied team section.
- **Highlights**: Recent blog posts and news.
- **Footer**: Comprehensive footer with links and social icons.

## Design System

### Fonts

The project uses the **CursorGothic** font family in various weights and styles:

- Regular
- _Italic_
- **Bold**
- **_Bold Italic_**

### Colors

The design utilizes a sophisticated dark theme palette:

**Backgrounds**

- Primary: `#14120b` (Deep Black/Brown)
- Secondary: `#1b1913` (Dark Grey/Brown)
- Light: `#222019` (Lighter accent background)
- Hover: `#201e17`

**Typography**

- **Primary Text**: `#e4dedc` (Off-white)
- **Secondary Text**: `#999795` (Muted Grey)
- **Accent**: `#e04c00` (Vibrant Orange)
- **Accent Hover**: `#b14005` (Darker Orange)

**UI Elements**

- **Buttons**: `#edecec` (Light Grey)
- **Borders**: `#3f3d38` and `#25241e` (Subtle dividers)
