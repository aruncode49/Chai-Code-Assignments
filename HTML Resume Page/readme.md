# HTML Resume Page

This is a simple personal resume website created using HTML. It shows information about a person's skills, experience, and education.

## How to View

To see the resume, download the zip file and extract it. Then simply **double-click** the `index.html` file in this folder. It will open in your web browser.

## Features

- **Header**: Shows name and title.
- **About Me**: A short introduction.
- **Skills**: List of technical skills.
- **Experience**: Work history in a table.
- **Projects**: Details about past projects.
- **Education**: School and degree information.
- **Contact**: Email and location.
